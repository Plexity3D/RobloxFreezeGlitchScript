# RobloxFreezeGlitchScript
Py Macro that does the roblox freeze glitch. (Works with patch)





## Setup:
1. Install: https://github.com/Plexity3D/RobloxFreezeGlitchScript/releases
2. Run freeze.exe
3. Yeah, that's it

## Using it:
Run Freeze.Py
press "Q" on the roblox bar! (That white bar above the roblox player)
Now you can press F3 to toggle it!


(Note: Breaks in first person/shiftlock)
